# AgentForge n8n Workflows

This folder contains exported workflow JSON files for import into n8n:

- `team-execution-flow.json` - baseline team execution and callback
- `retry-timeout-flow.json` - retry/timeout handling flow
- `notification-flow.json` - completion notification flow

## Import

1. Open n8n UI.
2. Choose **Import from file**.
3. Import the workflow JSON from this folder.
4. Adjust service URLs/environment for your deployment.
